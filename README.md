Project Title: Real-Time Analog Clock

Objective:
The objective of this project is to create a real-time analog clock that displays the current system time accurately. The clock runs concurrently and updates every second to reflect the correct hour, minute, and second hands, providing a visual and functional representation of time.

Features:
Real-time updating of hour, minute, and second hands
Smooth animation for clock hand movements
Responsive and clean UI design
Lightweight and runs directly in the browser
No page reloads required for updates

Technologies:
HTML & CSS for structure and styling
JavaScript for real-time functionality and logic
